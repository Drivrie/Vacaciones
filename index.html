☁️ Sincronización automática — GitHub Gist
Tus datos se guardan en un Gist privado de GitHub. Configura una vez y se sincroniza solo en todos tus dispositivos.
No conectado
Cómo configurar (1 sola vez):
1. Abre github.com/settings/tokens/new
2. Escribe un nombre: Wanderlust · Expiration: No expiration
3. En Select scopes marca únicamente ✓ gist
4. Pulsa Generate token y copia el token (ghp_...)
5. Pégalo aquí y pulsa Conectar — eso es todo
Solo necesitas el scope gist. El token se guarda únicamente en este dispositivo.
✅ Conectado — ¿Quieres usarlo en otro dispositivo?
Pasos en el otro dispositivo (iPhone, tablet…):
1. Abre esta misma página web en el otro dispositivo
2. Ve a Alertas → Sincronización GitHub Gist
3. Pega el ID del Gist (copia el de abajo)
4. Genera un nuevo token en GitHub con scope gist y pégalo
5. Pulsa ⬇ Cargar datos — los datos aparecen al instante
ID del Gist:
⚠ El token es personal de cada cuenta de GitHub. No lo compartas — cada dispositivo genera el suyo propio. El ID del Gist sí es compartible.
💡 Sincronización automática: Una vez configurado, los datos se guardan solos cada vez que haces un cambio y se cargan automáticamente al abrir la app. Si el estado muestra "⚠ error", pulsa ⬇ Cargar datos para forzar la carga.

🔄 Sincronizar entre dispositivos

Elige cómo quieres compartir tus datos entre ordenadores. Puedes usar más de una opción.

📦 JSON Fácil
Exporta todos tus datos a un archivo .json. Guárdalo en Google Drive / Dropbox y ábrelo en el otro ordenador con "Importar".
⚠ Esto reemplazará todos tus datos actuales.
Arrastra aquí el archivo .json
o haz clic para seleccionarlo
💾 HTML portable Recomendado
Genera una copia de este mismo archivo HTML con todos tus datos ya incrustados. Guarda ese archivo en Drive y ábrelo en cualquier ordenador sin instalar nada.
El archivo generado tendrá tus datos precargados. Cada vez que hagas cambios, genera uno nuevo.
☁️ GitHub Gist Recomendado
Sincronización automática en tiempo real con GitHub Gist. Los datos se guardan en la nube con cada cambio y se cargan solos al abrir en cualquier dispositivo. Solo necesitas un token con scope gist.
Ver estado arriba
Token necesario: solo scope gist (no "repo"). Se configura en la pestaña Alertas → Sincronización GitHub Gist.

📋 Código Apps Script para Google Sheets

Instrucciones:
1. Abre tu Google Sheets → Extensiones → Apps Script
2. Borra el código existente y pega el código de abajo
3. Guarda (Ctrl+S) y haz clic en Implementar → Nueva implementación
4. Tipo: App web · Acceso: Cualquier persona
5. Copia la URL generada y pégala en el campo del planeador
const SHEET_NAME = 'Wanderlust'; function doGet(e) { const action = e.parameter.action; if (action === 'load') return loadData(); return ContentService.createTextOutput('{"error":"unknown action"}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const data = JSON.parse(e.postData.contents); if (data.action === 'save') return saveData(data.payload); return ContentService.createTextOutput('{"error":"unknown action"}') .setMimeType(ContentService.MimeType.JSON); } function loadData() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) return ContentService.createTextOutput('{"data":null}') .setMimeType(ContentService.MimeType.JSON); const val = sheet.getRange('A1').getValue(); return ContentService.createTextOutput(JSON.stringify({data: val})) .setMimeType(ContentService.MimeType.JSON); } function saveData(payload) { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) sheet = ss.insertSheet(SHEET_NAME); sheet.getRange('A1').setValue(JSON.stringify(payload)); sheet.getRange('A2').setValue(new Date().toISOString()); return ContentService.createTextOutput('{"ok":true}') .setMimeType(ContentService.MimeType.JSON); }
